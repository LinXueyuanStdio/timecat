# timecat
时光猫本体开源计划

自从时光猫彻底组件化后，时光猫的开源计划开始执行。

由于部分代码涉及服务器安全、用户信息安全等，不能全部开源，但同学们对时光猫的功能实现呼声很高，所以打算选择了一些不敏感的功能进行开源，满足你们！

对希望开源的功能，请在下面的**功能提议**里投票决定。
